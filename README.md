# React lab 3 Project, product management
This is a React + TypeScript frontend application for managing products. The app connects to a REST API backend and allows users to view products, add new products, and open individual product pages using dynamic routing.

The project uses Context API for state management, JWT authentication for protected admin access, individiual product pages and React Router with createBrowserRouter for navigation.

The frontend is deployed on Netlify and communicates with a backend deployed on Render.

clone the repository
npm install
npm run dev
npm run build
